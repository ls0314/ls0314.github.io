《Efﬁcient Visual State Space Model for Image Deblurring》精读

发表于 2025-09-12 更新于 2025-09-16 分类于 论文精读 阅读次数： 本文字数： 1.3k 阅读时长 ≈ 2 分钟

EVSSM本质是基于mamba的一个去模糊视觉状态模型，在去模糊方面CNN和Transformer都各有缺点，用文章的原话来说CNN”由于卷积运算在空间上是不变的且是局部性的，因此它不能有效地建模全局和空间变化的信息，从而限制了其实现更好图像恢复的能力“。而Transformer”自注意力机制（即，缩放点积注意力）在token数量方面需要二次空间和时间复杂度，这在处理高分辨率图像时变得不可接受。即使有些策略来降低计算复杂度，但它们无法有效地表征长程依赖关系和非局部信息”。而恰好mamba恰好可以在较低的计算复杂度下保证有效的非局部信息，这一性质正好适合解决CNN和Transformer在去模糊方面的缺失。

《Efﬁcient Visual State Space Model for Image Deblurring》一文对mamba的核心改进就是在扫描中引入几何变换（翻转，转置）以更加高效的捕捉非局部信息。此举表现在架构中的EVS模块，为了能更好的转换EVS模块的特征，被本文作者又提出了一个改进的EDFFN模块，此模块对EVS模块的特征进行频域增强的处理，可以更好的保留想捕获的特征。

EVS模块

在上图中可以很清晰的看到EVS模块的数据流过程，其中最为核心的就是Reshape即几何变换模块和S6这个基于mamba的计算模块。

在EVSSM原文中也提到图像特征在每经过4个EVSS模块后将自动恢复到原始空间结构。

EDFFN模块

这个模块就是对DFFN的魔改版本，值得一提的是DFFN也出自此团队，下面按照数据流的顺序对其中各模块介绍。第一个Conv1x1是一个卷积核为1x1的卷积神经网络，其作用是将数据的维度再扩大到二倍，为下方的门控处理做准备。Dconv是深度可分卷积神经网络，在保持shape不变的情况下进行局部空间混合使数据内包含局部空间感受野信息。GEGLU门控机制，将数据分为两半，一半进行非GLU线性变换另一半不进行处理，在将二者相乘以达到非线性和线性交互的目的。第二个Conv1x1，也是一个卷积核为1x1的卷积神经网络，负责将已经倍乘了的数据缩放回原先的shape。第一个Patch Unfolding 分块操作方便进行后续的傅里叶变换。FFT，傅里叶变换将数据转换为频域。 W为可训练的参数矩阵。 IFFT，逆向傅里叶变换将处理后的频域在转换回实域。第二个Patch Unfolding将处理好的patch拼回完整矩阵并剪切掉多余部分。

这一模块的主要目的是对特征的非线性部分进行增强。

EVSS模块

EVSS模块主要就是EVS + EDFFN 再加上两个归一化层和残差连接构成，再EVSSM的构造中的共有6个EVSS模块，按功能可以分为三类，与下采样搭配，与上采样搭配和单纯的EVSS模块 共同构成了完整的EVSSM
